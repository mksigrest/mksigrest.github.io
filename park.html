Country API

Role: Project Manager and Programmer

Languagues Used: HTML, Javascript

Programs Used: Visual Studio Code, Heroku, Github

What is the purpose of this application: This application is my own API, meant to be able to find countries based off of capitals, location, and financial information. It allows you to add countries to the API list locally, and search for them once added. You are able to search with HEAD or GET, returning heading or full information respectively. You can also edit a countries capital with the final node.

What was the design process:

What did I learn in this design process: I learned how to deploy a site on Heroku, and how to connect the github actions to that deployment. I learned about get and head node requests, and how turn a JSON file into an API service.
